# Kanban Board View Examples

## Overview

This document provides production-quality mockups of Kanban board views across all four rendering levels of the SaaSkit design system. The board displays a "Tasks" resource with columns: **Backlog**, **Todo**, **In Progress**, **Review**, and **Done**.

---

## Data Model Reference

```tsx
<App name="tasks">
  <Task
    title
    description?
    status="backlog | todo | in_progress | review | done"
    priority="low | medium | high | critical"
    assignee->User?
    tags:string[]
  />
  <User name email avatar? />
</App>
```

---

## Keyboard Hints

```
TASKS BOARD                                                    [?] Help

Navigation:
  Arrow keys / hjkl    Move between cards and columns
  Tab / Shift+Tab      Jump between columns
  Home / End           First / last card in column
  g g / G              First / last card on board

Actions:
  Enter                Open card details
  Space                Start dragging selected card
  n                    New card in current column
  e                    Edit selected card
  d                    Delete selected card (with confirm)
  m                    Move card (shows column picker)

Filtering:
  /                    Search cards
  f                    Open filter menu
  1-5                  Quick filter by priority
  @ + name             Filter by assignee

Other:
  r                    Refresh board
  ?                    Toggle this help
  q                    Quit / close board
```

# CLI Usage

```bash
# Interactive kanban board (default: unicode)
$ saaskit tasks board

# Specific rendering levels
$ saaskit tasks board --plain      # Level 1
$ saaskit tasks board --ascii      # Level 2
$ saaskit tasks board --unicode    # Level 3 (default)
$ saaskit tasks board --terminal   # Level 4

# Output formats
$ saaskit tasks board --json       # JSON for automation
$ saaskit tasks board --markdown   # For documentation

# Filtering
$ saaskit tasks board --assignee=mike
$ saaskit tasks board --priority=high
$ saaskit tasks board --tag=frontend

# Quick actions
$ saaskit tasks move 7 review      # Move task #7 to review
$ saaskit tasks create --column=todo --title="New task"
```
